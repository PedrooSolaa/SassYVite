# P3.1 Maquetación con Vite y SASS

> S2DAW Diseño Web
> Autor: Pedro Sola

## 1. Descripción

Este proyecto consiste en crear una página web responsive utilizando **SASS** y **Vite**. El objetivo es implementar un diseño previamente realizado en **Figma**, adaptándolo tanto para escritorio como para dispositivos móviles. Se aplican conceptos avanzados de SASS, como variables, mixins, anidamiento y media queries.
